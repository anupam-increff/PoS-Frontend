TSV Schema Information

Upload a Tab-Separated Values (TSV) file with the following columns:

clientName - Name of the client/company
barcode - Unique product barcode
name - Product name/description
mrp - Maximum Retail Price (numeric)
imageUrl - Product image URL (optional)
Note: First row should contain column headers. Values should be tab-separated.
